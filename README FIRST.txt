===================================================
Project: Robotic Arm Kinematics using GA
===================================================

# Requirements
Run this command to install dependencies:
pip install numpy matplotlib streamlit

# How to Run

Method 1: Web Interface (Streamlit)
- Double-click: Run_Web.bat
- (Or run in terminal: streamlit run "Robotic Arm Kinematics GA.py")

Method 2: Command Line (Local)
- Double-click: Run_Local.bat
- (Or run in terminal: python "Robotic Arm Kinematics GA.py")
- Note: Close the plot window to enter new inputs or exit.
